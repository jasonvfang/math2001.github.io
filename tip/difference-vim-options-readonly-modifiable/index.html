'readonly' != 'modifiable'

  • infos
  • vim
  • options

Do you know the difference between those 2 options

set readonly
set nomodifiable

All the readonly does, when enabled, is it prevents you from writing the file (you can bypass it by doing :w! though), while modifiable, when disabled prevents you from actually editing the buffer.
